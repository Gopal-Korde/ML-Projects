# House Price Prediction using Machine Learning

This project predicts house prices using multiple regression models and provides a web interface using Flask.

## Features
- Multiple ML models (Linear, Ridge, Lasso, Random Forest, XGBoost, LGBM, ANN, etc.)
- Model comparison table
- User-friendly Flask web application

## Input Features
- Average Area Income
- Average Area House Age
- Average Area Number of Rooms
- Average Area Number of Bedrooms
- Area Population

## Tech Stack
- Python
- Flask
- Pandas, NumPy
- Scikit-learn
- XGBoost, LightGBM

## How to Run
```bash
pip install -r requirements.txt
python app.py
