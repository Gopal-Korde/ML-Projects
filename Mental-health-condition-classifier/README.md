# Mental Health Condition Classifier

## Overview
A supervised machine learning project to predict mental health conditions using diagnostic data.

## Tech Stack
- Python
- Scikit-learn
- Pandas
- Matplotlib
- Seaborn

## Models Used
- Logistic Regression
- Support Vector Machine (SVM)
- Random Forest

## Accuracy
Achieved ~87% accuracy.

## How to Run
1. Install dependencies
2. Run `main.py`
